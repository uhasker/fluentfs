hofs.paths.paths.dir_exists¶

hofs.paths.paths.dir_exists(path: str) → bool¶

Check whether a directory is present at the given path.

Parameters:

path – The given path.

Returns:

True, if a directory is present. False if the path does not exist at all or if the path represents a (regular) file.
